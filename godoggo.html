Godoggo


The Problem

To design a responsive web app for location-based recommendations that meets the requirements of its users and solves the problems they face with existing location-based recommendation apps


The Solution

I decided that the user I would design for would be someone interested in taking their dog places, and making sure those places were dog-friendly. By honing in on a niche need, I was able to design solutions for this specific problem.

The Design Process

User Personas

Through conducting in depth user interviews, user needs were identified. After analyzing the interview responses and prioritizing their goals, personas were created for the three key user profiles.

User Flows

From the user personas, Jobs to be done were identified and an MVP document was created. The MVP identified five key functions for the first iteration of the product. A user flow was created to map out how users would move through the web app.

Low-Fidelity Prototype

The user flow was then used to create the first round of low-fidelity wireframes. Low-fidelity wireframes were drawn using a Crazy 8’s process, to try out multiple screen layouts. Once the favorite screens were selected from the drawn wireframes, a low-fidelity prototype was created to test the user flow.

Tester feedback from the paper prototype was reviewed and adjustments were implemented to create the first round of digital wireframes.

Moodboard

Through researching design patterns and visual inspiration, a mood board was created as the jumping off point for the style guide. The mood board was used as a “north star” to stay within a cohesive visual style.

Style Guide

Typography

The primary font used for all headers and copy is “Montserrat” in various weights. “Dongle” should only be used for as an accent font for buttons and inputs.

Brand Colors

The prominence of each brand color should reflect their size relation to each other shown at the right. The five primary brand colors are Forest Green, Sun Beam, Snowcaps, Deep Teal and Teal Foam. Secondary colors, Soft Moss, Mustard, and Cabernet, should only be used selectively to complement the other colors.

Icons

Icons for godoggo should use soft edges to mimic the soft curves of the watermark logo. Icons use 2 pixel stroke width in combination with coordinating fill, to relate to the filled in paw print icon used in our logo.

Logo

The godoggo logo is made up of three colors from our core palette. When the logo is used on a light ground, it should be comprised of these colors. If the godoggo logo is used on a dark background, it should be simplified using the Snowcaps color only, as shown. The logo can be used larger than shown, however the minimum padding on all four sides must be at least the same size as the height of the logo. The logo must never be shown in any other colors, and should never be pixelated.

Imagery

Like our brand, our imagery should show dogs and their owners on adventures. Photography should evoke feelings of being in nature, and experiencing the outdoors. Slightly more saturated images with dark greens and earthy tones reinforce the adventurous lifestyle of our users. Dogs should always be shown enjoying places in the real world, cafes, parks, trails, restaurants and more. The images should make you want to go to the places shown and experience new things.

Copy Language

Language used in copy should convey a playful sense of adventure, always reminding our users that of their active and fun-loving lifestyle.
